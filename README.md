# Governance decisions

This repository contains formal decision records that are relevant to the
governance, design, and evolution of the Open Education API (OEAPI).

The purpose of this repository is to provide a single, stable location for
decisions that are:

- not version-specific
- not temporary or exploratory
- intended to have lasting impact across the ecosystem

## Scope

This repository may include decision records related to:

- architecture and design choices
- governance and organisational agreements
- data modelling principles
- implementation directions and constraints
- cross-cutting concerns that apply to multiple versions

Version-specific changes or short-lived discussions are out of scope and should
be handled in the relevant specification or documentation repositories.

## Structure

Decision records are stored as individual files using a consistent structure.
Each record should:

- have a clear and descriptive title
- include context, decision, and consequences
- be immutable once accepted
